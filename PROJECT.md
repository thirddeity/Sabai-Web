# Sabai Web Project

## Project Summary

Sabai Web is a Personal Digital Assistant for Thai users. The app helps one person keep important life information in one simple place.

The first goal is to build a small, usable MVP for personal use before expanding to a wider audience.

## Product Direction

- Simple enough for people with little technical background.
- Friendly for Thai users of many ages.
- Low budget and low maintenance.
- Safe handling of personal data.
- Built one task at a time.
- Repository files are the source of truth.

## MVP Modules

1. Digital Vault
2. Finance Tracker
3. Schedule & Reminder
4. Task Management
5. Dashboard
6. Simple Search
7. Simple In-app Reminder

## MVP Exclusions

Do not add these to the MVP:

- AI
- Payment
- Team workspace
- Complex admin panel
- Docker Compose
- Caddy
- VPS
- Redis
- Background queue
- Microservices
- Kubernetes

## Working Rules

- Keep the system as small as possible.
- Do not scaffold or build features before the project control documents are ready.
- Do not rely on old chat history.
- Read the project documents before every implementation task.
- Update `PROGRESS.md` and `ROADMAP.md` before finishing a task.
- Update `DECISIONS.md` when a new project decision is made.
